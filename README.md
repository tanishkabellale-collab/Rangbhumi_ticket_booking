# Rangbhumi RSCOE Ticket Booking Mini Project

A Python Flask mini project to book college natak tickets with 2D seat selection and PDF ticket generation (with QR code). 

---

## **Features**

- Landing page with 3 shows and countdown timers.
- 2D seat selection with:
  - Color coding: VIP (yellow), booked (red), available (green)
  - Click or drag to select multiple seats
  - Dynamic price calculation
  - Suggest Best Seat button
- Simulated payment page
- PDF ticket generation with QR code
- Downloadable ticket with seat details
